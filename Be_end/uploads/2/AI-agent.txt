1. AI Workflow Builder(轻量版 Dify/Coze)
热度:极高,AI Agent 编排平台是当前最热赛道之一。

RAG:用户上传文档,构建知识库,节点可检索
Tool-Use:内置工具节点(搜索、代码执行、API 调用),用户可组合
Workflow:可视化拖拽画布编排 Agent 工作流,节点连线,条件分支
亮点:前端用 Vue 做拖拽画布非常出彩,面试/展示效果极好

Vue Flow 集成要点:

使用 @vue-flow/core 的 VueFlow 组件

通过 useVueFlow() composable 管理节点/边状态

自定义节点通过 nodeTypes 选项注册

拖拽放置通过 onDrop 事件 + addNodes 实现
